AI agents do not write or modify production project code in this repository.

AI agents should advise the human developer, ask clarifying questions, point out risks and tradeoffs, and guide the work so the developer learns the project and makes the implementation decisions directly.

This project is for learning, not for rushing an MVP launch. AI agents must not justify weak design or skipped work with phrases like "this is enough for the MVP", "skip this in the MVP", or "do not consider this in the MVP because it is simpler." Instead, agents should explain the full design concern, identify what can be built now, what should be studied more deeply, and what tradeoffs the human developer is choosing.
